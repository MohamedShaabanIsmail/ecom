üìã Overview

This API provides comprehensive endpoints for managing products in an e-commerce platform. It supports CRUD operations, advanced filtering, pagination, product search, and image management with Redis caching support.

Version: 1.0.0 | Framework: Spring Boot 3.4.2 | Java: 21

üîå API Endpoints

GET Get All Products

/api/products

Retrieve a paginated list of products with optional filtering by brand, price range, and category.

Query Parameters:

Parameter Type Required Default Description
pageNum Integer No 1 Page number (1-indexed)
pageSize Integer No 5 Products per page

Request Body (Optional):

{ "brand": "Nike", "minPrice": 50.00, "maxPrice": 200.00, "category": "Electronics" }

Response (200 OK):

[ { "id": 1, "name": "Nike Air Max", "description": "High-performance running shoe", "brand": "Nike", "price": 149.99, "category": "Footwear", "releaseDate": "2025-11-16T00:00:00.000+00:00", "productAvailable": true, "stockQuantity": 50, "imageName": "nike_air_max.jpg", "imageType": "image/jpeg", "imageData": null } ]

GET Get Product by ID

/api/product/{id}

Retrieve detailed information about a specific product.

Path Parameters:

Parameter Type Description
id Integer Product ID

Response (200 OK):

{ "id": 1, "name": "Nike Air Max", "description": "High-performance running shoe", "brand": "Nike", "price": 149.99, "productAvailable": true, "stockQuantity": 50 }

Status 404: Returned if product not found

POST Create Product

/api/product

Create a new product with image upload (multipart/form-data).

Form Data:

Field Type Required Description
product JSON Yes Product object
imageFile File Yes Image file (JPG, PNG, etc.)

Response (201 Created):

Returns the created product with ID and image data

Status 500: Returned if file upload fails

GET Get Product Image

/api/product/{id}/image

Retrieve the image file for a specific product.

Path Parameters:

Parameter Type Description
id Integer Product ID

Response (200 OK):

Binary image data with appropriate Content-Type header (image/jpeg, image/png, etc.)

GET Search Products

/api/products/search

Search for products by keyword (searches name, description, and brand).

Query Parameters:

Parameter Type Required Description
keyword String Yes Search keyword

Response (302 Found):

Array of matching products
